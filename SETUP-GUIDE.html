🚀 RentMyStuff Backend Setup Guide

Follow these 6 steps in order. Each step builds on the previous one.

1
Install Node.js on your computer
Start here
▼

Node.js is the engine that runs your backend server. You need version 18 or higher.

⬇ Download Node.js (LTS)

After installing, open Terminal (Mac/Linux) or Command Prompt (Windows) and verify:

node --version # should show v18.x or higher npm --version # should show 9.x or higher
⚠️ If you see "command not found" — restart your terminal after installing Node.js
2
Set up the project and install packages
5 min
▼

Extract the ZIP file you downloaded. Open terminal inside the rentmystuff-backend folder and run:

npm install # installs express, mongoose, razorpay, etc.

You'll see a node_modules folder appear. That means it worked. ✅

⚠️ If you get errors, make sure you're inside the correct folder — it should have package.json in it
3
Create free MongoDB Atlas database
10 min
▼

MongoDB Atlas is a free cloud database — no credit card needed.

🍃 Open MongoDB Atlas

Follow these steps inside Atlas:

  • Click "Try Free" and create an account
  • Choose "Create a cluster" → select M0 Free tier
  • Pick any cloud region (Mumbai is fastest for India)
  • Wait ~2 minutes for the cluster to be created
  • Click "Connect" → "Connect your application"
  • Copy the connection string — looks like: mongodb+srv://username:password@cluster0.xxxxx.mongodb.net/
  • Go to Network Access → Add IP Address → Allow from Anywhere 0.0.0.0/0
  • Go to Database Access → Add user with username & password
💡 In your connection string, replace <password> with the actual password you set. Also add rentmystuff as the database name before the ?

Your final MONGO_URI should look like:

mongodb+srv://rahul:mypassword123@cluster0.abc12.mongodb.net/rentmystuff?retryWrites=true&w=majority
4
Get Razorpay test API keys
5 min
▼

Razorpay gives you test keys to develop with — no real money moves until you go live.

💳 Create Razorpay Account

Get your test keys:

  • Log in to Razorpay Dashboard
  • Go to Settings → API Keys
  • Make sure you're in Test Mode (toggle at top right)
  • Click "Generate Test Key"
  • Copy both Key ID and Key Secret
💡 Test keys start with rzp_test_. Use these during development — they won't charge real money. Switch to live keys only when you're ready to launch.

Your keys will look like this:

RAZORPAY_KEY_ID=rzp_test_AbCdEfGh12345678 RAZORPAY_KEY_SECRET=xYzAbCdEfGhIjKlMnOpQrSt
5
Create your .env file
5 min
▼

The .env file holds all your secret keys. Never share this file or commit it to GitHub.

Inside your project folder, create a file named exactly .env (with the dot) and paste this:

# Server PORT=5000 NODE_ENV=development # MongoDB — paste your connection string here MONGO_URI=mongodb+srv://youruser:yourpassword@cluster0.xxxxx.mongodb.net/rentmystuff?retryWrites=true&w=majority # JWT Secret — any long random string JWT_SECRET=rentmystuff_super_secret_key_2025_change_this JWT_EXPIRE=30d # Razorpay keys (from dashboard.razorpay.com) RAZORPAY_KEY_ID=rzp_test_your_key_id_here RAZORPAY_KEY_SECRET=your_razorpay_secret_here # Frontend URL FRONTEND_URL=http://localhost:3000
⚠️ Replace every placeholder with your real values from Steps 3 and 4
6
Verify connections and start the server
Final step!
▼

First, run the connection checker to make sure everything is set up correctly:

node check-connection.js

You should see all ✅ green. If anything shows ❌, it will tell you exactly how to fix it.

Once everything is green, start the server:

npm run dev # starts server + auto-restarts on changes

You should see:

🚀 RentMyStuff server running on http://localhost:5000 ✅ MongoDB connected successfully

Then run the API tests to verify everything works:

node test-api.js
💡 Open http://localhost:5000 in your browser — you should see the API welcome message
💳 How the Razorpay payment flow works
1. Renter clicks Pay
→
2. Backend creates order
→
3. Razorpay popup opens
→
4. User pays (UPI/card)
→
5. Backend verifies signature
→
6. Booking confirmed + OTP sent
The signature verification in Step 5 is critical — it prevents fake payment confirmations. Razorpay signs each payment with your secret key, and our backend verifies it using HMAC-SHA256 before confirming the booking.
